The projects below provide Finitio implementation in various host languages. Contact us or fork this page on github to add your own binding to this list!

Finitio for Ruby

The finitio ruby gem provides a mature implementation in Ruby. You'll also find the source code on github:

  • blambeau/finitio-rb

finitio-rb implements the full specification of Finitio 0.3.0.

Finitio for Javascript

The finitio.js project is under active development for providing Finitio to Javascript developers. The source code is available on github:

  • llambeau/finitio.js

finitio.js is about to release, and conforms to Finitio 0.2.0.
